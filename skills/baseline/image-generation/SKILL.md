# Image Generation Skill

Create diagrams, charts, and visual assets.

## Capabilities

- **Diagrams**: Generate network and architecture diagrams
- **Charts**: Create data visualizations and graphs
- **Flowcharts**: Build process and workflow diagrams
- **Infographics**: Design visual summaries

## Status

This skill is planned for development. See the [docx skill](../docx/) for an example of a completed baseline skill.

## Planned Features

- Network topology diagrams
- Attack path visualizations
- Risk matrices and heatmaps
- Timeline graphics
- Org charts and hierarchy diagrams
- Data flow diagrams
- Bar charts, pie charts, trend lines
- Export to PNG, SVG, PDF

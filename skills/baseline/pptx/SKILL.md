# PPTX Skill

Read, modify, and create Microsoft PowerPoint presentations.

## Capabilities

- **Read Presentations**: Extract text, images, and notes from PPTX files
- **Create Presentations**: Generate new presentations from templates
- **Modify Presentations**: Edit existing slides, add content
- **Template-Based Generation**: Use corporate templates for consistency

## Status

This skill is planned for development. See the [docx skill](../docx/) for an example of a completed baseline skill.

## Planned Features

- Create slides from markdown content
- Apply corporate templates and branding
- Generate executive summary slides
- Add charts and diagrams
- Insert tables and formatted text
- Create security briefing presentations
- Convert assessment findings to slides

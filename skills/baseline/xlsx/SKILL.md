# XLSX Skill

Read, modify, and create Microsoft Excel spreadsheets.

## Capabilities

- **Read Spreadsheets**: Extract data, formulas, and formatting from XLSX files
- **Create Spreadsheets**: Generate new Excel files with data and formatting
- **Modify Spreadsheets**: Edit existing workbooks, add/remove sheets
- **Data Export**: Export data to Excel format from various sources

## Status

This skill is planned for development. See the [docx skill](../docx/) for an example of a completed baseline skill.

## Planned Features

- Read worksheets and extract cell values
- Write data to cells with formatting
- Create charts and pivot tables
- Apply conditional formatting
- Work with named ranges
- Import/export CSV data
- Generate security metrics dashboards
